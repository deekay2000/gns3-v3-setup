# gns3-v3-setup
Skripte für die automatisierte GNS3 V3 Installation,
gehört zur Bachelorarbeit "Einrichtung und Erprobung der 
Netzwerksimulationssoftware GNS3 in der neuen 
Version 3 mit Mehrclient-Betrieb auf einem externen 
Server" von David Kinzel

## proxmox_gns3_setup:
Dient dazu, eine GNS3-VM mit der aktuellen Version 3.X.X zu ersetellen.

## gns3_v3_pool_patch:
Dient dazu, die Datenbankauslegung für den Multiuserbetrieb anzupassen.
